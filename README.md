# CI/CD Pipeline with GitHub Actions and OpenShift Tekton

## Project Overview
This project demonstrates a complete CI/CD pipeline using:
- GitHub Actions for Continuous Integration
- Tekton Pipelines on OpenShift for Continuous Delivery
- Python application with flake8 linting and nose unit testing

## Tools Used
- GitHub Actions
- Tekton Pipelines
- OpenShift
- Python
- flake8
- nose

## Pipeline Stages
1. Linting with flake8
2. Unit testing with nose
3. Build and deploy using Tekton
4. Persistent storage using OpenShift PVC
